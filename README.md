## 🚀 Deploy

File ini di-host via **GitHub Pages** dengan custom domain `adeitiar.com`

## 📁 Struktur File

```
/
├── index.html   ← Website utama (upload ini)
├── CNAME        ← Custom domain config
└── README.md    ← Dokumentasi
```

## ⚙️ Cara Upload ke GitHub

1. Buat repository baru di github.com
2. Upload semua file ini
3. Settings → Pages → Branch: main → Save
4. Pasang custom domain di Settings → Pages → Custom domain
